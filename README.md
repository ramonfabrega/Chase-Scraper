# Chase-Scraper

`git clone https://github.com/ramonfabrega/Chase-Scraper/`

## Usage

One liner run

`USERNAME="$user" PASSWORD="$pass" node run balance.js`

---

Alternatively,

create `credentials.js` at root and input info

```
module.exports = {
  username: $user,
  password: $pass'
};
```

and run with `node run balance.js`
